# AI-based-voice-calculater

My project is an AI-based Voice Calculator developed using Python and Flask. It understands both text and voice commands to perform mathematical calculations. The system uses a Hidden Markov Model (HMM) to identify the required mathematical operation from the user's input. It supports basic operations such as addition, subtraction, multiplication, and division, along with advanced functions like square root, factorial, percentage, power, sine, cosine, tangent, cotangent, secant, and cosecant. The calculator automatically extracts numbers from natural language sentences and performs the correct calculation. It provides results on a user-friendly web interface and also reads them aloud using text-to-speech. The application stores previous calculations in a history section for easy access. This project combines Artificial Intelligence, Natural Language Processing, and Web Development to create a smart and interactive calculator. It is useful for students, teachers, engineers, professionals, and people with visual impairments or physical disabilities, as they can perform calculations easily using voice commands.
